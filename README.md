# PostgreSQL Analytics Project

## Overview
This project demonstrates analytical reporting on sales, products, and customers using PostgreSQL. It includes schema creation, data population, queries for insights, and reporting views.

## Project Structure
- `schema.sql` : Database and schema setup.
- `tables/` : Scripts to create dimension and fact tables.
- `queries/` : Analytical queries for sales, products, and customer analysis.
- `views/` : Predefined views for product and customer reporting.

## Key Features
1. Sales analysis over time (yearly, monthly, running totals, moving averages)
2. Product performance and segmentation
3. Customer segmentation (VIP, Regular, New) and behavior analysis
4. Category contribution analysis
5. Ready-to-use reporting views for dashboards
